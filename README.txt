# Bioelectrical impedance analysis (BIA) Dashboard Visualizer

This project is for those who frequently monitor their body status using bioelectrical impedance analysis (BIA) but find it challenging to interpret .csv formatted data. With this project, you can easily visualize how your diet, training, and health are progressing.

## About
This web application enables you to create visual representations of the following factors:
1) Weight in kg/lb
2) Fat-Mass (FM) in kg/lb and %
3) Fat-Free-Mass (FFM) in kg/lb and %
4) Intra-Cellular-Weight (ICW%)
5) Extra-Cellular-Weight (ECW%)
6) Basal Metabolic Rate (BMR)

The application will display these data based on their respective ExamDate.

## Getting Started
You can use this project on the following link: https://bia-dashboard-visualizer.streamlit.app/
In the web application, you can download a "Correctly Formatted Dataset" to see how your data needs to be structured in order for the application to work properly.

## Launch Locally
If you want to launch the app locally, please make sure to install all the prerequisites first. Once you have all you need, simply open 1_BIA-Dashboard.py in your IDE and just run the code. The app will be launched locally.
